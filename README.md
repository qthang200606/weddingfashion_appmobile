# 📱 Overview

Wedding Fashion Mobile App is a full-featured Android application developed using **Kotlin** and **Jetpack Compose**.

The application provides a seamless shopping experience for customers looking for wedding dresses, suits and wedding accessories. It integrates Artificial Intelligence to support customers with personalized product consultation and recommendations.

Besides customer features, the system also provides an administrator dashboard for managing products, appointments, orders and inventory.
